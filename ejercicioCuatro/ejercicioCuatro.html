Cuarto Ejercicio

Descripción:

Considere que se tienen dos arreglos, x, y, del mismo tipo de datos entre ambos. La tarea es crear un programa que imprima los elementos en común si es que existen. La forma de los arreglos es:

x = [x1,x2,...,z1,...,zk,...,xn]

y = [y1,y2,...,z1,...,zk,...,ym]

Note que el conjunto de elementos, (z1,...,zk), representa la intersección entre el arreglo x e y, donde debe de entenderse que 0 ≤ k ≤ min(n,m).

Sea P() el programa de interés entonces este debe de comportarse de la forma:

P(x,y) = (z1,...,zk)
